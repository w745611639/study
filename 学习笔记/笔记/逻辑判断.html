宽松相等

下面就 x == y 进行说明

1.字符串和数字的比较

1.如果Type(x) 是数字, Type(y) 是字符串, 那么返回 x == ToNumber(y) 的结果

2. 如果Type(x) 是字符串, Type(y) 是数字， 那么返回 ToNumber(x) == y 的结果

2.其他类型和布尔值进行比较

1.如果Type(x) 是布尔值， 那么返回 ToNumber(x) == y的结果

2.如果Type(y) 是布尔值， 那么返回 x == ToNumber(y)的结果

3. null 和 undefined 进行比较

1.如果x为null，y为undefined 那么返回true

1.如果x为undefined，y为null 那么返回true

关于null 和 undefined 注意

					
						var a = null;
						var b;
						a == b       			// true
						a > 0   				// false
						a < 0   				// false
						a == 0       			// false
						a >= 0 					// true
						a <= 0 					// true
						b < 0  					// false
						b > 0   				// false
						b == 0    				// false
						b >= 0					// true
						b <= 0    				// true

从上面代码我们可能会有疑问， 为什么 null >= 0 null <= 0 是true呢

其实可以这么看 null >= 0 就是 null < 0的结果取反

对象和非对象之间进行比较

1.如果Type(x) 是字符串或者数字，Type(y) 是对象, 那么返回 x == ToPrimitive(y) 的结果

1.如果Type(x) 是对象，Type(y) 是字符串或者数字, 那么返回 ToPrimitive(x) == y 的结果

布尔值会首先转换成数字
